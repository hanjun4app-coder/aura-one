# AURA ONE - INTERACTION RULES

## Official Gesture Language

Wave Left
→ Previous Product

Wave Right
→ Next Product

Palm Hold
→ Open Product

Fist
→ Collapse Scene

Palm Release
→ Particle Burst

Hover Hand
→ Focus Object

## Rules

- AI must not invent new gesture meanings without approval.
- Every gesture must have clear visual feedback.
- Avoid gesture overload.
- Gestures should feel natural and easy to remember.
- If gesture recognition is uncertain, system should stay stable instead of triggering random actions.
