# AURA ONE - VISUAL LANGUAGE

## Style Direction

AURA ONE should feel:
- premium
- cinematic
- spatial
- futuristic
- elegant
- calm
- immersive
- minimal

## Avoid

- cyberpunk overload
- gamer UI
- chaotic particles
- excessive glow
- noisy animation
- too many effects
- low-end sci-fi aesthetics

## Motion Principles

Motion should be:
- smooth
- intentional
- responsive
- readable
- calm
- premium

## Spatial Principles

The scene should have:
- depth
- breathing room
- clear focus
- cinematic composition
- readable interaction feedback
